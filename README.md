##Main Window Function

1. 오늘 날짜와 시간을 출력해준다
2. "Today"버튼 - 오늘 날짜를 표시해준다
3. 각 날짜버튼을 누르면 일정을 추가할 수 있는 대화상자가 뜬다.
4. * '<<' : 작년 날짜로 보여줌
   * '>>' : 내년 날짜로 보여줌
   * '<'  : 지난달 날짜로 보여줌
   * '>'  : 다음달 날짜로 보여줌
